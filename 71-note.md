
if / else 문은 복잡한 논리를 위해 함께 연결할 수 있습니다. 다음은 다중 체인 if / else if 문의 의사 코드입니다.

if (조건 1) {
  statement1
} else if (조건 2) {
  statement2
} else if (조건 3) {
  statement3
. . .
} else {
  statementN
}
다음 조건을 충족하기 위해 연결된 if / else if 문을 작성합니다.

num <5- "Tiny"반환
num <10- "Small"반환
num <15- "Medium"반환
num <20- "Large"반환
num> = 20- "Huge"반환
