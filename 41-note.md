
배열 끝에 데이터를 추가하는 쉬운 방법은 push () 함수를 사용하는 것입니다.

.push ()는 하나 이상의 매개 변수를 가져 와서 배열의 끝에 "푸시"합니다.

예 :

var arr1 = [1,2,3];
arr1.push (4);
// arr1은 이제 [1,2,3,4]입니다.

var arr2 = [ "Stimpson", "J", "cat"];
arr2.push ([ "행복", "기쁨"]);
// arr2는 이제 [ "Stimpson", "J", "cat", [ "happy", "joy"]]와 같습니다.
