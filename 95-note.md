루프를 사용하여 동일한 코드를 여러 번 실행할 수 있습니다.

가장 일반적인 유형의 JavaScript 루프는 특정 횟수 동안 "for"실행되기 때문에 for 루프라고합니다.

For 루프는 세미콜론으로 구분 된 세 가지 선택적 표현식으로 선언됩니다.

for ([초기화]; [조건]; [최종 표현])

초기화 문은 루프가 시작되기 전에 한 번만 실행됩니다. 일반적으로 루프 변수를 정의하고 설정하는 데 사용됩니다.

조건문은 모든 루프 반복이 시작될 때 평가되며 true로 평가되는 한 계속됩니다. 반복 시작시 조건이 거짓이면 루프 실행이 중지됩니다. 즉, 조건이 거짓으로 시작하면 루프가 실행되지 않습니다.

final-expression은 다음 조건 검사 전에 각 루프 반복의 끝에서 실행되며 일반적으로 루프 카운터를 늘리거나 줄이는 데 사용됩니다.

다음 예제에서는 i = 0으로 초기화하고 조건 i <5가 참인 동안 반복합니다. 최종 표현식으로 i ++를 사용하여 각 루프 반복에서 i를 1 씩 증가시킵니다.

var ourArray = [];
for (var i = 0; i <5; i ++) {
  ourArray.push (i);
}
ourArray는 이제 [0,1,2,3,4]를 포함합니다.
